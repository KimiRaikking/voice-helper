① 它能做什么

  • 全局可用:任何能接受粘贴的输入框(终端、浏览器、微信、备忘录、邮件……)都能用。
  • 按住说话:按住「右 Option」键录音,松开即转写并自动粘贴。
  • 多引擎可切换:Whisper(多语言)/ SenseVoice(中文优化),VOICE_ENGINE 一键切换,模型都在本地、互不替换。
  • 状态可见:顶部菜单栏图标实时显示状态,并标注当前引擎。
  • 强制简体中文:偶尔输出繁体时自动转简体(不影响英文)。
  • 开机自启:后台服务,开机自动运行、崩溃自动重启。
例外:密码输入框(macOS 安全键盘机制屏蔽模拟粘贴)、全屏独占键盘的 App 可能用不了。

③ 日常使用

  1. 把光标放进任意输入框。
  2. 按住「右 Option」键 → 说话 → 松开。
  3. 文字自动转写并粘贴到光标处。

菜单栏图标(屏幕最顶部系统栏)实时显示状态;点开菜单可见当前引擎([Whisper]/[SenseVoice])、最近识别文字(点击可复制)、退出按钮。

图标含义
🎤待命
🔴录音中
⏳转写中
✅完成(闪一下回到 🎤)
⚠️太短 / 失败

④ 识别引擎

通过 VOICE_ENGINE 选择,两个引擎模型都在本地,可随时切换:

引擎VOICE_ENGINE特点适用
Whisperwhisper(默认)多语言,中英混说友好通用 / 英文多
SenseVoicesensevoice阿里达摩院,中文/方言优化,带智能标点,非自回归极快中文为主
关于准确度:中文「技术同音词」(时延 / 实验 / 食言)对所有通用模型都是难点——会默认挑最常见的词,换模型只能小幅缓解。真正对症的是热词偏置(见第⑩节)。

⑤ 配置项(环境变量)

环境变量作用默认
VOICE_ENGINE识别引擎:whisper / sensevoicewhisper
VOICE_KEY热键:alt_r/alt_l/ctrl_r/cmd_r/f5…alt_r(右Option)
VOICE_LANG强制语言 zh/en;空=自动空
WHISPER_MODELWhisper 用的模型whisper-large-v3-turbo
SENSEVOICE_MODELSenseVoice 用的模型iic/SenseVoiceSmall
VOICE_PROMPTWhisper 专用,热词/上下文提示一组技术词
VOICE_SOUND设值=开声音提示关
VOICE_NOTIFY设 1=完成弹通知关
VOICE_NO_PASTE设值=只复制不粘贴关

Whisper 模型选项(速度 vs 精度)

模型大小速度精度
whisper-small-mlx~0.5GB很快中
whisper-large-v3-turbo ←默认~1.6GB快(~0.8s)高
whisper-large-v3-mlx~3GB稍慢最高

当前生效配置(plist)

VOICE_KEY=alt_r
VOICE_ENGINE=sensevoice
VOICE_LANG=zh
VOICE_PROMPT=<一组技术热词>

⑥ 如何修改配置

open -e ~/Library/LaunchAgents/com.zhanggang.voiced.plist

在 EnvironmentVariables 的 <dict> 里增删,例如切回 Whisper:

<key>VOICE_ENGINE</key>
<string>whisper</string>

改完重启服务生效:

launchctl kickstart -k gui/$(id -u)/com.zhanggang.voiced

⑦ 开机自启 / 服务管理

后台常驻靠 macOS LaunchAgent:RunAtLoad=true(登录即启动)、KeepAlive=true(崩溃自动重启)。

# 查看是否在运行
launchctl list | grep voiced

# 重启(改完代码或配置后)
launchctl kickstart -k gui/$(id -u)/com.zhanggang.voiced

# 临时关闭 / 重新开启
launchctl unload ~/Library/LaunchAgents/com.zhanggang.voiced.plist
launchctl load   ~/Library/LaunchAgents/com.zhanggang.voiced.plist

# 实时看日志
tail -f ~/voice-claude/voiced.log

⑨ 常见问题

按了没反应,日志报 PortAudioError ... [PaErrorCode -9986]?
录音设备状态变了(插拔耳机、切换音频输出、麦克风被占用)导致 PortAudio 缓存失效。重启服务即可恢复:

launchctl kickstart -k gui/$(id -u)/com.zhanggang.voiced

菜单栏看不到图标? 在屏幕最顶部系统栏(不是终端里),刘海可能挡住。按住热键说话时盯顶栏看图标变化即正常。

按键没反应 → 缺「输入监控」;有识别但没粘贴 → 缺「辅助功能」;录不到音 → 缺「麦克风」。授权后均需重启服务。

输出繁体字? 已用 OpenCC 强制转简体;若仍出现,检查 opencc 是否在 venv 里。

改了 voiced.py 不生效? 需 launchctl kickstart -k ... 重启。

下载 Whisper large-v3 卡在 0.00B? HF 大文件改用了 Xet 后端,某些网络下会卡死。禁用 Xet 走旧通道:

HF_HUB_DISABLE_XET=1 ~/voice-claude/.venv/bin/python -c \
  "from huggingface_hub import snapshot_download; print(snapshot_download('mlx-community/whisper-large-v3-mlx'))"

该模型权重文件名为 weights.npz(~3GB)。

⑩ 规划中:热词偏置(技术词准确度)

针对「技术同音词」(时延/实验、推理/椎离)的根治方案是热词偏置:
  • 引擎换成 FunASR 的 SeACo-Paraformer(支持 hotword 参数)
  • 新增 VOICE_ENGINE=paraformer 与 VOICE_HOTWORD=<词表>
  • 模型从 ModelScope 下载(国内快),funasr 已就绪
  • 把高频技术词/专有名词放进 VOICE_HOTWORD,遇同音歧义强制优先匹配
尚未启用。启用后 Whisper / SenseVoice / Paraformer 三引擎可自由切换。
